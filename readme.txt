== Description ==

Need to display your Twitter feed? Then you can't live without this plugin, easily display your tweets anywhere on your site, with Title, Username & Tweet Limit options.

== Installation ==

1. Upload the /ct-twitter/ folder to the /wp-content/plugins/ directory.
2. Activate the CT Twitter plugin through the "Plugins" menu in WordPress.
3. Configure your settings via Appearance > Widgets > CT Twitter.
4. And you're good to go!
